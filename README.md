# Описание
Проект на базе npm.

Используются postCSS и handlebars.

Все исходники лежат в /src

### Дев сервер 
Используется vite.

Комманда для запуска: **npm run dev**

### Продакшен сервер
Используется express для отдачи файлов. Для сбоки: vite. 

Запуск: **npm run start**
